UPI Transaction Analysis Dashboard – Power BI

An interactive Power BI dashboard designed to analyze UPI (Unified Payments Interface) transactions. The project focuses on data cleaning, transformation, and visualization to uncover insights into balances, payments, device usage, city trends, and customer demographics.

--Features
Dynamic Filters & Slicers – filter by Bank, City, Device, Gender, Merchant, etc.
Charts & Visuals – line and column charts for balances and transaction trends
Bookmark Navigator – seamless report navigation
Age Group Segmentation – created a calculated column (using IF condition)
Data Cleaning – handled in Power Query Editor (duplicates, missing values, validation)

--Skills & Tools
Power BI
Power Query
DAX
Data Cleaning
Data Visualization & Storytelling

--Outcome

A user-friendly dashboard that:

Monitors UPI transaction patterns and balances
Provides insights into customer behavior
Analyzes demographics & geographies for better decision-making
